Class ParametriRicercaAssenza

Declaration :

  • Java : class ParametriRicercaAssenza extends ParametriRicerca

Identificativo classe: CSW-01/17

Contiene i criteri per ricercare le assenze.

Attribut idStudente

Declaration :

  • Java : private long idStudente

Id dello studente che ha fatto l'assenza

Attribut giustificata

Declaration :

  • Java : private boolean giustificata

Specifica se l'assenza è giustificata o meno

Attribut dataInizio

Declaration :

  • Java : private date dataInizio

Giorno/ora di inizio dell'assenza

Attribut dataFine

Declaration :

  • Java : private date dataFine

Giorno/ora di fine dell'assenza

Attribut data

Declaration :

  • Java : private date data

Indica una data coperta dall'assenza.

Per esempio, se si specifica il valore 05/02/2007, la ricerca può
recuperare un'assenza che inizia il 01/02/2007 e finisce il 10/02/2007,
dato che la data specificata rientra nel periodo dell'assenza.
