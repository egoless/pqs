Class WSAssenza

Declaration :

  • Java : interface WSAssenza

Identificativo classe: CSW-01/20

Web Service che offre metodi per la manipolazione delle assenze.

Operation aggiungiAssenza

Declaration :

  • Java : public long aggiungiAssenza()

Aggiunge una nuova assenza di un certo tipo associata ad uno studente. Il metodo restituisce l'id che verrà assegnato all'assenza.

Operation modificaAssenza

Declaration :

  • Java : public boolean modificaAssenza()

Modifica una assenza esistente.

Operation cancellaAssenza

Declaration :

  • Java : public boolean cancellaAssenza()

Cancella una assenza con un dato id.

Operation cercaAssenza

Declaration :

  • Java : public Collection cercaAssenza()

Restituisce un insieme di assenze che rispettano i criteri definiti in ParametriRicercaAssenza. Si può decidere se ogni assenza ha tutti i dati o solo quelli più significativi.

All public operations : aggiungiAssenza , cancellaAssenza , cercaAssenza , modificaAssenza
